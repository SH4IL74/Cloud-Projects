# ☁️ Cloud-Projects Workspace
A centralized monorepo for Cloud Engineering and DevOps projects, focusing on AWS automation and Infrastructure as Code (IaC).

## 📁 Project Structure
- **[aws-cloud-infra/](./aws-cloud-infra/)**: Infrastructure as Code using **Terraform** to provision AWS resources.
- **[cloud-dev/](./cloud-dev/)**: AWS automation and management scripts using **Python** and **Boto3**.

## 🛠️ Tech Stack
- **Cloud Provider:** Amazon Web Services (AWS)
- **Local Development:** LocalStack, Docker
- **Automation:** Python 3.x, Boto3
- **Infrastructure:** Terraform
- **Environment:** WSL (Ubuntu)

## 🚀 How to Use
1. Clone the repository: `git clone https://github.com/SH4IL74/Cloud-Projects.git`
2. Navigate to a project folder (e.g., `cd cloud-dev`).
3. Follow the specific instructions in each subfolder's README (coming soon).
